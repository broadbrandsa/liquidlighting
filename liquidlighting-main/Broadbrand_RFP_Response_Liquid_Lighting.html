6.3 Field Sales Mobile App (React Native)

Delivered by: Digitalise

Purpose: A dedicated mobile application for sales representatives to create professional quotes on-site at customer premises, trade shows, and project sites - even without internet connectivity.

Core Features

  • Cross-Platform: Single codebase for iOS and Android using React Native
  • Offline-First Architecture: Full functionality without internet connection, automatic sync when back online
  • Tablet & Phone Optimised: Responsive interface designed for both form factors

Product Catalogue

  • Full product browsing with images, specifications, and pricing
  • Advanced search and filtering by category, type, finish, wattage, etc.
  • Barcode/QR scanning to quickly add products
  • Product favourites and recently used items
  • Cached catalogue for offline access with smart sync
  • Real-time stock availability (when online)

Quote Creation

  • Intuitive quote builder with drag-and-drop line items
  • Room/area-based project organisation
  • Quick quantity adjustments and variant selection
  • Tiered pricing based on customer type (retail/trade)
  • Flexible discounting (line item, overall, trade pricing)
  • Notes and annotations per line item
  • Quote duplication and templates for common configurations

Customer Management

  • Customer lookup from CRM database
  • Create new customer/prospect on the fly
  • Customer visit history and previous quotes
  • Contact details with click-to-call/email
  • GPS check-in for site visits (optional)

Quote Delivery

  • Professional PDF quote generation on-device
  • Email quote directly to customer from app
  • WhatsApp share integration
  • Digital signature capture for quote acceptance
  • Convert accepted quote to order

Sales Tools

  • Product comparison tool
  • Lighting calculation helper (lumens, coverage area)
  • Photo capture and attachment to quotes (site photos)
  • Technical data sheets and brochures (offline available)
  • Installation guides and specifications

Sync & Integration

  • Background sync with central systems
  • Conflict resolution for offline edits
  • Push notifications for quote approvals/updates
  • Integration with Shopify, QContact CRM, and backend systems
  • Activity logging for sales management reporting

Security

  • Biometric authentication (Face ID / Fingerprint)
  • PIN/password protection
  • Remote wipe capability
  • Role-based access control
  • Encrypted local data storage

Admin & Reporting

  • Sales activity dashboard (quotes created, conversion rates)
  • Rep performance tracking
  • Quote pipeline visibility
  • Export reports to management dashboard

10. Integration Architecture

10.1 Integration Approach

We will develop a central integration layer (middleware) that connects all systems, providing:

  • Single source of truth for data mapping
  • Error handling and retry logic
  • Logging and audit trails
  • Real-time and scheduled sync options
  • Scalable architecture for future integrations

10.2 Data Flows

Data Type Source Destination Frequency
